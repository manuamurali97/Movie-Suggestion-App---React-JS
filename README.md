# Movie Explorer App

A React-based movie browsing application powered by the TMDB API. The project includes a movie search feature, display of movie details through cards, and a favorites system built using the Context API with localStorage persistence.

Features:

-> Search movies using live data from the TMDB API

-> Display movie posters, titles, and release years

-> Add and remove favorite movies

-> Persistent favorites stored in localStorage

-> Responsive and clean user interface

-> Modular and maintainable component structure

Technology Stack:

React (Vite or Create React App)

Context API

TMDB API

CSS

---- Favorites System Overview ----

The favorites functionality is implemented using the React Context API. Favorite movies are stored in state and synchronized with localStorage to allow persistence across page reloads. Clicking the favorite button on any movie card toggles its status.

---- API Reference ----

This project retrieves movie data from The Movie Database (TMDB):

https://www.themoviedb.org/
